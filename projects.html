Selected Projects

This page highlights selected technical and engineering projects related to robotics, autonomous systems, and applied machine learning.

Multirotor Drone Fleet Platform

Designed, assembled, and maintained a multi-vehicle multirotor drone fleet for laboratory-scale robotics and control experimentation.

Tools: CAD, 3D printing, power electronics, PX4/ArduPilot, soldering

Autonomous Egg Collection Robot

Developed a wheeled mobile robot capable of autonomously collecting and dumping eggs using a custom mechanical dumping mechanism.

Tools: Mechanical design, actuation, embedded control

Ambient Sound Recognition

Implemented a supervised machine learning pipeline for ambient sound classification, including feature extraction, model training, and evaluation.

Context: Machine Learning course project
Tools: Python, scikit-learn, signal processing

Wall-Following F1TENTH Autonomous Vehicle

Implemented a wall-following controller on an F1TENTH autonomous vehicle platform using LiDAR-based perception and real-time control logic.

Context: Autonomous Vehicle course project
Tools: ROS, Python/C++, LiDAR
